# 🎓 Student Performance Analysis

A Streamlit-based web application for managing and analyzing student marks using Python, Pandas, Matplotlib, and Seaborn.

## 🚀 Features

* Add New Students
* Delete Students
* View Student Records
* Find Topper Automatically
* Subject-wise Performance Analysis
* Dynamic Bar Charts
* Interactive Heatmap Visualization
* CSV-Based Data Storage

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas
* Matplotlib
* Seaborn

## 📂 Project Structure

```text
student-performance-analysis
│
├── app.py
├── students.csv
├── requirements.txt
└── README.md
```

## 📊 Visualizations

* Maths Score Analysis
* Subject Average Comparison
* Student Performance Heatmap

## ▶️ Run Locally

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
